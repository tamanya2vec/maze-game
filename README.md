# 🌀 Maze Game

Welcome to my Scratch maze game! Navigate through the walls to reach the end.

### 🎮 How to Play
* Use the **Arrow Keys** or **WASD** to move.
* Avoid the red wall, black wall and the mermaid!
* Reach the green portal to win the level.
